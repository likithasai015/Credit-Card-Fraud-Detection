# 💳 Credit Card Fraud Detection

## 📌 Overview

This project detects fraudulent credit card transactions using Machine Learning. It uses classification models to distinguish between normal and fraudulent transactions.

---

## ⚙️ Technologies Used

* Python
* Scikit-learn
* Pandas, NumPy
* Streamlit

---

## 🚀 Features

* Detects fraud transactions
* Handles imbalanced dataset
* Web interface using Streamlit
* Real-time prediction

---

## 📊 Model Performance

* Algorithm: Random Forest
* High ROC-AUC Score (~0.99)
* Good precision and recall

---

## 🌐 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
